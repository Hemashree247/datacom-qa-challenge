## 1. Project Setup & Execution

### Prerequisites
* Node.js (LTS version)
* npm (installed with Node.js)

### Setup
1.  Clone this repository: `git clone https://github.com/Hemashree247/datacom-qa-challenge.git`
2.  Navigate to the project directory: `cd datacom-qa-challenge`
3.  Install dependencies: `npm install`
4.  Install Playwright browsers: `npx playwright install`

### Run Tests
Execute all automated tests:
```bash
npx playwright test
